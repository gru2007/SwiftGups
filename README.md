## 📖 Использование

### Первый запуск

1. При первом запуске откроется экран приветствия
2. Заполните профиль: имя, факультет, группа
3. Выберите группу из списка или найдите через поиск
4. При необходимости откройте `Профиль` → `Авторизация ДВГУПС` и подключите личный кабинет
5. Начните пользоваться приложением!

### Расписание

- **Выбор недели**: используйте кнопки "←" и "→" или календарь
- **Поиск группы**: введите название в поле поиска
- **Обновление**: потяните вниз для обновления расписания
- **Live Activity**: включите в настройках профиля

## ⚠️ Известные ограничения

1. **Зависимость от внешнего API**: приложение полностью зависит от доступности сайта `dvgups.ru`
2. **Авторизация расписания**: при сбросе серверной сессии приложение автоматически переавторизуется только если сохранены логин и пароль ЛК ДВГУПС
3. **Live Activity**: требует iOS 16.1+ и физическое устройство для полной функциональности
4. **Background Tasks**: ограничены системой iOS (не гарантируется выполнение)
